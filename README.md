# Visionz App

Visionz is an online "vision board" that allows users to upload inspiring images along with descriptions as well as browse images
that were uploaded by other users> This is a place to inspire and get inspired.


## Getting started

To get started with the app, clone the repo and then install the needed gems:

```
$ bundle install --without production
```

Next, migrate the database:

```
$ rails db:migrate
```
Run the app in a local server:

```
$ rails server
```

You are now ready to create a profile and begin browsing or uploading your own images.
